Motivational Message Generator
A simple, lightweight web application that generates random motivational messages at the click of a button.

Overview
This project is a front-end web application built with pure HTML, CSS, and JavaScript. It randomly combines text fragments to create unique, uplifting motivational quotes for the user.

Features
Zero Dependencies: Built using vanilla HTML, CSS, and JavaScript.
Dynamic Content: Generates a new, random message every time the user clicks the “Get Motivational Message” button.
Simple UI: A clean, centered layout for focused user interaction.
How to Run
Ensure you have all the files in the same directory:
index.html
MotivationalMessage.js
Open the index.html file in any modern web browser.
Click the “Get Motivational Message” button to see your daily dose of motivation.
Code Structure
index.html: Defines the user interface, including the button and the display container.
MotivationalMessage.js: Contains the logic for the app. It holds arrays of starters, messages, and endings, and uses a generateMessage() function to construct and display the final sentence.
Technologies Used
